# Allplan BIM Documentation Library

This repository contains technical documentation for Allplan BIM software, organized into logical categories for easier access and reference. Duplicate content has been removed to provide a more streamlined and efficient resource.

## Categories

### 1. BIM Technical Reference Guides
Comprehensive technical documentation covering advanced Allplan BIM functionality, including:
- Open BIM workflows
- Advanced modeling techniques
- Automation features
- Structural detailing
- Infrastructure and terrain modeling
- IFC data exchange processes
- Attribute mapping techniques

**Files in this category:**
- Allplan-BIM-Comprehensive-Technical-Reference-Guide.txt

### 2. IFC Data Exchange
This category is now empty after consolidating all IFC-related content into the main technical reference guide.

### 3. User Guides
General user guides and instructional materials for Allplan software:
- Basic to advanced user tutorials
- Interface navigation
- Feature explanations
- Best practices

**Files in this category:**
- Allplan-BIM-Complete-User-Guide.txt
- Allplan-Comprehensive-User-Guide.md *(Enhanced with detailed technical information, best practices, Allplan 2026 features, precast detailing module, and real-world case studies from reference documents)*
- Allplan-Guide-Summary.md

### 4. Menu Navigation
This category is now empty. The interface navigation document was removed during the consolidation process.
